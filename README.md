# pytest-rs

pytest-rs is a re-implementation of the popular Python testing framework [pytest](https://github.com/pytest-dev/pytest) in Rust.

> Note: This project is currently in active development (alpha stage). Many features are still under implementation and subject to change. Documentation, including installation and usage instructions, will be provided in future releases.

## Planned for Alpha Release

- Test Execution
    - test functions
    - test files
    - test directories
    - async test funcion support
- Fixture Resuolution
  - geneerator fixture
  - async fixture
- Assert replacement
